# seqlib

Library for handling multiple sequencer tracks, focused on generative sequencers.

Used in Microlidian, Nexus6 and Compulidian.

Currently:-
 - Mutating, modulatable Euclidian sequencers
 - Turing Machine-esque

Uses the [midihelpers](https://github.com/doctea/midihelpers), [parameters](https://github.com/doctea/parameters), [mymenu](https://github.com/doctea/mymenu), [saveloadlib](https://github.com/doctea/saveloadlib) libraries.
